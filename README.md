# start writing in rust in 2021

## enviroment

1. install `rust`
2. use `nightly`
3. vscode extension `rust-analyzer`
4. mirror use `https://mirrors.sjtug.sjtu.edu.cn/git/crates.io-index`
5. write in rust every day.


## use

- build: `cargo build`
- run: `cargo run --bin xxx`(eg. `cargo run --bin log_test`)
